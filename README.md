# Darts Pro 2016

## Desktop version: 
  1. Download the folder Darts from our repository.
  2. Then just start the .exe file which is in the folder: Darts/Darts/bin/Debug/Darts.exe

## Web version:
  1. Download the folder Web_Darts from our repository.
  2. Then you need server. If you are on Windows you can install xampp or some application like xampp. Link to download xampp: https://www.apachefriends.org/index.html . If you are on Linux then you can install Apache or xampp again. Link to download Apache - https://httpd.apache.org/download.cgi .
  3. After installing one of the servers now you can proceed.
  4. What to do on xampp:<br/>
    4.1. First you need to start the application. Go to the folder where you install your xampp and there must be some file like this - xampp-control.exe. When you start it the following window should pop up: http://imgur.com/xHnU6XG .<br/><br/>
    4.2. After you can see this window, you need to click the button Start on Apache and MySQL. And you should see something like this: http://imgur.com/tTfTe2p . If you have problem with starting Apache you should know that Skype use the ports, which Apache need. So you can go to Skype Settings, then go to Expanded Settings and there to Connection. Then remove the tick from "Use port 80 and 443". Save and try again. If you have again problem with starting apache you can GOOGLE your problem or connect with us to try to solve your problem. Our mail: dartspro2016@gmail.com.<br/><br/>
    4.3. Once you start all these things you can close this window and do the following:<br/> <br/>
      4.3.1. First go to the download directory from the beggining. Copy all files from this folder and paste them into xampp/htdocs/<br/><br/>
      4.3.2. After you done this just time in url: localhost/ and you should see our application.<br/><br/>
